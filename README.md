# Coworking Prilep — coworkingprilep.mk

Official website for **Coworking Prilep**, the first coworking space in Prilep, North Macedonia.

- **Live (planned):** https://coworkingprilep.mk
- **Status:** pre-launch landing — full site in development
- **Opening:** 2026

## Stack

- [Astro](https://astro.build/) (^6.x)
- React 19 (only where interactivity is needed)
- Tailwind CSS 4
- Deployed to Coolify on the personal VPS

## Local development

```bash
npm install
npm run dev
```

Then visit `http://localhost:4321`.

## Build

```bash
npm run build
npm run preview
```

## Project location

Source lives in iCloud Drive so it syncs across the two Macs:

```
~/Library/Mobile Documents/com~apple~CloudDocs/CoworkingPrilep/
```
